# PM BY MADI — премиальный лендинг перманентного макияжа

Современный адаптивный сайт для мастера перманентного макияжа «PM BY MADI», созданный на Next.js 14 c Tailwind CSS. Включает SEO-оптимизированный контент, интерактивную галерею, отзывы, FAQ и форму записи с отправкой заявки в WhatsApp.

## Ключевые блоки
- Главный экран с УТП и CTA-кнопками
- О мастере, услуги и преимущества
- Портфолио с подгрузкой работ и отзывами в карусели
- FAQ и подробный блок записи с формой
- Контакты с картой, социальными сетями и мессенджерами

## Технологии
- Next.js 14 (App Router)
- React 18, Tailwind CSS
- next/font для подключения шрифтов
- Данные в JSON для удобного редактирования контента

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Полезные команды

```bash
npm run build   # Сборка production-версии
npm start       # Запуск production-сборки
npm run lint    # Проверка ESLint
```

## Структура проекта
- `app/` – маршрутизация и страницы
- `components/` – переиспользуемые блоки и UI-компоненты
- `data/` – контент сайта (услуги, FAQ, отзывы и пр.)
- `public/` – изображения и статичные ассеты

## Деплой

Проект готов к публикации на Vercel или любой другой платформе для Next.js:
1. Убедитесь, что сборка проходит: `npm run build`.
2. Закоммитьте изменения и запушьте в GitHub.
3. Импортируйте репозиторий в Vercel (настройки по умолчанию).

## Настройка под себя
- Обновите контактные данные и ссылки в `data/site.json` и `data/links.json`.
- Скорректируйте цены, описания услуг и FAQ в соответствующих файлах в `data/`.
- Замените изображения в папке `public/` на свои работы, сохранив названия или обновив пути в `data/gallery.json`.
